Liquidpad

Programmable Open Source Macropad

The Liquidpad is a customizable macropad developed by Alex Symons and Varun Sreedharan. The Liquidpad is based on the popular and powerful Raspberry Pi Pico development board. With the Liquidpad, the user will be able to enhance their computing experience by storing keyboard macros and passwords on the Liquidpad, saving time and increasing productivity. The Liquidpad is currently in the beta prototype stage of it’s development. After we finish finalizing and polishing the firmware, board design, case design, and configuration app we hope to release the project on Kickstarter. Even though the product will be available for purchase, we will still keep all the design files and software public, so that repair and modification of the Liquidpad is made as simple as possible. Creating products that are unrepairable, through lack of documentation and schematics goes against the design philosophy of Fish Island. We hope to help pave the way for a new paradigm of hardware and software development.

This github repository for the project files: https://github.com/FishIslandElectronics/Micropython-Macropad
